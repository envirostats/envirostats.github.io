Welcome

R Language for Statistical Computing in Business Analysis

Hello and welcome to our course website! My name is Stefan Schreiber and I am your course instructor for EXGEN 5449. The room number is ENT 2 590G and the course runs from Friday to Sunday from 9:00 am to 4:30 pm.

In this course, I will provide you with an introduction to the R programming language to perform data manipulation, data visualization and basic statistical analyses. Since this will be an introductory course, you won’t need any prior experience in R, however, if you do, it will certainly be helpful. I designed the course to be hands-on, which means I will refrain from providing lecture-type presentations but instead spend time with you writing R code and answer questions as they appear. As with any other language, at first, it may seem impossible to ever get good at it but with time and practice you will become a data science wizard.

For this course, we will be working with R, R Studio and the tidyverse packages. If you feel like it, you can visit those links and install R and R Studio on your laptops (but we will do this together in class as well!). That being said, for this course you need a laptop, R, R Studio, a web browser, and a spreadsheet editor such as Excel (or a simple text editor will work too).

I also recommend two books to serve as reference guides. The first one is called R for Data Science, which also serves as our coursebook, and the second one is Hands-On Programming with R. You can read both books online for free, or purchase a hard-copy from Amazon if you prefer.

As of now, I am planning to cover the following topics (but I am flexible and can adapt to the needs of you guys!):

Course Overview

Friday (Day 1)

  • Intro to R (Downloading R, Getting to know R, R Scripts, Data structures)

  • Intro to R Studio (Downloading R Studio, Exploring R Studio)

  • General introduction to statistics

    • Descriptive statistics

    • Probability distributions (binomial, Poisson, normal)

    • Inferential statistics

    • Statistical assumptions

Saturday (Day 2)

  • Intro to the tidyverse packages

  • Reading and creating tidy data using the tidyr package

  • Data manipulation using the dplyr package

  • Data visualization using the ggplot2 package

  • String manipulation using the stringr package

  • Basic statistical analyses

    • Correlation

    • Linear regression

    • Analysis of variance

    • Generalized linear models

  • Producing tidy model outputs using the broom package

Sunday (Day 3)

  • Presenting your results using R Studio and R Markdown

  • Your turn! Pick a dataset (your own, or an R dataset), read it into R, wrangle it, visualize it, model it, communicate it.

  • General trouble shooting

See you soon!
